Stock Market Intelligence Suite

The Stock Market Intelligence Suite is a comprehensive data analytics project designed to help users understand stock behavior, evaluate financial performance, and make informed investment decisions. It uses Excel, Power BI, and Tableau to transform raw stock market data into meaningful insights through modeling, dashboards, and visual analytics.
